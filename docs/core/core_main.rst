Modeling in RDFBones
====================

This section describes the modeling principles of RDFBones and the ontologies from which RDFBones draws concepts from. This section gives a more in-depth explanation on some of the concepts and rules mentioned in the introductory guide and gives insight into aspects not covered in the introduction, especially aspects regarding the creation of ontology extensions.
